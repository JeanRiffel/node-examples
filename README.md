# node-examples

Esse repositório é para estudos dos principais pontos a respeito do NodeJS.


### Principais comandos


- npm init:  criar o arquivo package.json

- npm install nodemon --save   --save-dev 

- npm remove nodemon 

- npm install

- *diretório node_modules

- colocar no .gitignore
